Cppreference:MachineTranslations

From cppreference.com

Cppreference.com is a complex website. Hopefully you won't need to learn much of the complexity in order to successfully improve the machine-generated translations.

[edit] Basic process

The machine-translated snippets contain both the translated and original text. In wikicode, they look something like this: 

{{tr| machine-translated text | original text}}

We only need to retain the translated text -- as a translator, you only need to replace this entire construct with a proper translation of original text. The translation of the entire snippet should look like this: 

verified translation

Note that none of {{tr|, | or }} tokens are left. The original text is also removed.

After all text within an article is translated, please remove the translation notice ({{tr note}} text at the top of the article.

[edit] Previous translations

Some articles have old versions that have been overwritten with machine-translated versions because their layout was incompatible (the wiki has had several radical changes in the past). These versions might contain useful information that can be found by viewing the page history.

[edit] Templates

Most of the articles in the wiki are assembled from small snippets from a variety of pages called templates. Most of these templates can be edited via [edit] links that appear near where they are used.

[edit] Bots

Some very frequently used phrases (e.g. those used in headings) might have been incorrectly translated. In such cases, a bot could be used to fix the translations. Any such requests should be made here.

[edit] Help

If you have any questions regarding translations, you can ask here or here.
